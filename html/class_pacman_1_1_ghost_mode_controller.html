Detailed Description

Controls ghost mode timing (Chase/Scatter waves).

Original Pac-Man pattern (Level 1): Scatter(7s) -> Chase(20s) -> Scatter(7s) -> Chase(20s) -> Scatter(5s) -> Chase(20s) -> Scatter(5s) -> Chase(forever)

◆ Update()

void Pacman::GhostModeController::Update ( float deltaTime)

Updates the internal timers and switches modes when necessary.

The logic follows standard Pac-Man rules:

  1. If Frightened, decrement that timer only (Wave timer is paused).
  2. If in Permanent Chase (end of level logic), do nothing.
  3. Otherwise, increment Wave timer.
  4. If Wave timer exceeds duration, switch mode (Scatter <-> Chase) and advance wave count.

    deltaTimeTime elapsed since the last frame (in seconds).
