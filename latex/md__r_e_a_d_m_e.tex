A small Pacman-\/style game implemented in C++ using SFML and CMake. The repository contains the application source under {\ttfamily GUI/} and {\ttfamily Logic/}, assets under {\ttfamily assets/}, and CMake configuration files for building on multiple platforms.\hypertarget{md__r_e_a_d_m_e_autotoc_md999}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md999}

\begin{DoxyItemize}
\item Simple 2D game using SFML
\item CMake-\/based cross-\/platform build
\item Unit tests available under {\ttfamily Tests/}
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md1000}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md__r_e_a_d_m_e_autotoc_md1000}
