Provide a string representation for a given scancode.    

The returned string is a short, non-\/technical description of the key represented with the given scancode. Most effectively used in user interfaces, as the description for the key takes the users keyboard layout into consideration.

\begin{DoxyWarning}{Warning}
The result is OS-\/dependent\+: for example, \doxylink{structsf_1_1_keyboard_1_1_scan_aa42fbf6954d6f81f7606e566c7abe875ad71eadd79e28f8532df041dacc6302de}{sf\+::\+Keyboard\+::\+Scan\+::\+LSystem} is "{}\+Left Meta"{} on Linux, "{}\+Left Windows"{} on Windows and "{}\+Left Command"{} on mac\+OS.
\end{DoxyWarning}
The current keyboard layout set by the operating system is used to interpret the scancode\+: for example, \doxylink{classsf_1_1_keyboard_acb4cacd7cc5802dec45724cf3314a142ac5c87a05fd4d9d297402eb5d0d70a830}{sf\+::\+Keyboard\+::\+Semicolon} is mapped to "{};"{} for layout and to "{}é"{} for others.
