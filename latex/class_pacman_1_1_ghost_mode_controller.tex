Updates the internal timers and switches modes when necessary. 

The logic follows standard Pac-\/\+Man rules\+:
\begin{DoxyEnumerate}
\item If Frightened, decrement that timer only (Wave timer is paused).
\item If in Permanent Chase (end of level logic), do nothing.
\item Otherwise, increment Wave timer.
\item If Wave timer exceeds duration, switch mode (Scatter \texorpdfstring{$<$}{<}-\/\texorpdfstring{$>$}{>} Chase) and advance wave count.
\end{DoxyEnumerate}
