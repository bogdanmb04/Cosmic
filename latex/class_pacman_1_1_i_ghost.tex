Implements the Strategy Pattern. The game engine uses this interface to request the next target, regardless of the specific ghost type.

Calculates the specific target tile during Chase mode. 

This logic varies for each ghost\+:
\begin{DoxyItemize}
\item Red\+: Targets player directly.
\item Pink\+: Targets 4 tiles in front of player.
\item Blue\+: Uses a vector based on Red\textquotesingle{}s position and Player\textquotesingle{}s position.
\item Orange\+: Targets player, but retreats to scatter corner if too close.
