\doxysubsection{Detailed Description}
Defines a system event and its parameters. 

\doxylink{classsf_1_1_event}{sf\+::\+Event} holds all the informations about a system event that just happened. Events are retrieved using the \doxylink{classsf_1_1_window_base_a6a143de089c8716bd42c38c781268f7f}{sf\+::\+Window\+::poll\+Event} and \doxylink{classsf_1_1_window_base_aa1c100a69b5bc0c84e23a4652d51ac41}{sf\+::\+Window\+::wait\+Event} functions.

A \doxylink{classsf_1_1_event}{sf\+::\+Event} instance contains the type of the event (mouse moved, key pressed, window closed, ...) as well as the details about this particular event. Please note that the event parameters are defined in a union, which means that only the member matching the type of the event will be properly filled; all other members will have undefined values and must not be read if the type of the event doesn\textquotesingle{}t match. For example, if you received a Key\+Pressed event, then you must read the event.\+key member, all other members such as event.\+mouse\+Move or event.\+text will have undefined values.
