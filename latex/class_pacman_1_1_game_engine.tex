\doxysubsection{Detailed Description}
The central brain of the application. 

Inherits from \doxylink{class_pacman_1_1_i_game_engine}{IGame\+Engine} to provide a stable API for the GUI. Handles thread safety using std\+::mutex since the UI and Logic might run on different threads.

Performs the following steps\+:
\begin{DoxyEnumerate}
\item Updates \doxylink{class_pacman_1_1_ghost}{Ghost} Mode timers (Scatter/\+Chase/\+Frightened).
\item Checks for mode changes to reverse ghost directions.
\item Updates Player position based on accumulated time (fixed step).
\item Updates \doxylink{class_pacman_1_1_ghost}{Ghost} positions based on their specific speed interval.
\item Checks for collisions and Win condition. 
\end{DoxyEnumerate}
